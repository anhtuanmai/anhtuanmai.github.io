# MAI Anh Tuan — GitHub Pages Resume

Personal portfolio and resume site, live at [https://anhtuanmai.github.io](https://anhtuanmai.github.io) .

---

## Project structure

```
.
├── docs/                 # Jekyll build output — served by GitHub Pages
│   ├── index.html        # Main resume page (single-page static site)
│   └── assets/           # Images, icons, and static assets
├── .gitignore            # Ignores OS, editor, and local files
└── README.md             # This file
```

---

## Tech stack

- Pure **HTML5 / CSS3 / Vanilla JS** — no framework required
- **Jekyll** as the static site generator (GitHub Pages built-in support)
- **Fontshare** CDN for typography (Satoshi + Zodiak)
- Light and dark mode with system preference detection
- Fully responsive — mobile-first layout
